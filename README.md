# Standard Scientific Plotting Framework (SciPlot)

这是一个即插即用的出版级（如 _Nature_ / _Science_ / _IEEE_）标准科研绘图自动化脚本框架。

## 🎯 核心能力优势

1. **中英文字体分轨机制**：强制实现图表中文字体（宋体/SimSun）与外文、数字标点（Times New Roman）的正交混排，彻底杜绝特殊符号导致的降级排版。
2. **极简出图边框控制**：默认剥离单图上方和右侧的不必要向外延伸刻度线，并实现坐标轴全画幅闭合，使 Y 轴实现“首尾与极值等高”的 5 刻度均分约束。
3. **600 DPI 纯净导出**：出图分辨率从底层卡死在不低于 600 DPI，且统一为高防锯齿平滑的 `.png` 超高清位图。
4. **统计配套解析系统**：不仅仅只是绘制图表，框架现在会在出图的同时自动归纳出数据的样本数、均值、标准差等，可以直接无缝粘贴到报告论文内。

## 📁 核心结构指南

- **`/configs`**：存放如全局分辨率、单行拼图尺寸表与字号等宏观配置文件，使样式设置高度解耦于逻辑代码，供任何人即刻更改。
- **`/data`**：所有绘图的数据进出池。区分了绝对禁止修改的 `/raw` (元数据) 与计算落地的 `/processed` (过程清洗数据)，保障试验数据不被污染破坏。
- **`/scripts`**：由 `style_config` 与 `plot_utils` 构成的渲染引擎和功能控制中枢。
- **`/figures`**：高质量出版图专属的产出存放处，一目了然。
- **`/examples`**：内置多张实用的快速调用示例程序，看懂即可照搬。
